# react-boilerplate

### What is this?
Minimal React boilerplate

#### Stage 1 does:
* ES6 support
* JSX support
* Webpack Dev server with hot loading (changes are reflected in your browser as you press save)
* Linting with ESLint

### Install

```
npm install
```

### Running

```
npm start
open http://localhost:3000
```

### Linting

```
npm run lint
```
